# Public Repo Agent Instructions

## Scope

These instructions apply when working in the public assets repository.

## Purpose

This repo stores publishable and reusable assets only.

## Allowed Content

- Shared diagrams and editable sources
- Public prompts
- Public tool schemas
- Public eval cases, judges, and reports
- Public runnable examples
- Public reference indexes

## Not Allowed Here

- Private outlines
- Private editorial writing rules
- Draft articles that are not meant for release
- Article-specific private images
- Any material that requires reading the private repo to be understood

## Dependency Direction

- The private repo may reference this repo
- This repo must not reference private-only files as required inputs
