# edAgent-Public

这里收集的是可以长期复用的内容。

目前主要包括：

- 图示与可编辑源文件
- Prompt 模板
- 工具 schema
- 评测样本与结果
- 小型示例代码
- 参考资料笔记

## 目录

```text
.
├─ README.md
├─ CONTRIBUTING.md
├─ assets/
├─ prompts/
├─ tool-schemas/
├─ evals/
├─ examples/
└─ references/
```

## 说明

- 可编辑图源放在 `assets/diagrams/src/`
- 导出图片放在 `assets/diagrams/exports/`
- 可复用的 Prompt 放在 `prompts/`
- 工具定义放在 `tool-schemas/`
- 评测数据放在 `evals/`

命名和整理方式见 [CONTRIBUTING.md](CONTRIBUTING.md)。
